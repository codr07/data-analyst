# 📊 Customer Segmentation Analysis

This project demonstrates customer behavioral segmentation using RFM (Recency, Frequency, Monetary) analysis and K-Means Clustering. It extracts transactional records from an SQLite database and performs machine learning clustering in Python to classify customers into distinct business profiles.

## 🎯 Objectives
- Calculate RFM metrics using structured SQL aggregation queries on transactional data.
- Apply K-Means clustering to partition customers into behavioral segments.
- Provide targeted marketing insights for each segment based on features (Age, Income, Spending patterns).

## 🛠️ Tech Stack
- **Database**: SQLite
- **Data Engineering**: Python, SQLite3, Pandas
- **Machine Learning**: Scikit-learn (KMeans, StandardScaler)
- **Data Visualization**: Matplotlib

## 📈 Methodology & Pipeline
1. **Data Ingestion**: SQLite database structure containing two tables (`customers` and `purchases`).
2. **Feature Extraction (SQL)**: Aggregating transaction history to extract:
   - **Recency**: Days since the customer's last purchase.
   - **Frequency**: Total number of purchases.
   - **Monetary**: Total revenue generated by the customer.
3. **Data Preprocessing**: Standardize features using Z-score scaling.
4. **Clustering & Model Fitting**: Apply K-Means clustering.
5. **Profiling & Analysis**: Assign business segment labels based on centroid characteristics:
   - **VIP/Premium Shoppers**: High spending, moderate frequency, high income.
   - **Regular Value Shoppers**: Average spending, regular frequency, average income.
   - **Budget Shoppers**: Frequent small purchases, lower income, younger age.
   - **Dormant/Low-Value**: Joined long ago, low frequency, high recency (inactive).

## 🚀 How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Generate synthetic customer data:
   ```bash
   python generate_data.py
   ```
3. Run the RFM clustering and visualization pipeline:
   ```bash
   python analysis.py
   ```

## 📊 Key Deliverables
- `customer_data.db`: SQLite database populated with simulated customer profiles and transactional events.
- `customer_segments_summary.csv`: Aggregated cluster metrics detailing customer count, mean RFM, age, and income.
- `visualizations/customer_rfm_clusters.png`: 3D scatter plot of RFM segments.
- `visualizations/income_vs_spending.png`: 2D scatter plot showcasing income vs monetary spend by cluster.
